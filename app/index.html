Projects

Below are listed some projects that I participated during my academic and working period with a brief software description.
All projects finished while working are properties of the company.

Cool place finder

Issue REST call to public GeoNames api and WeatherBug api and filter the fetched data. Use quadtree data structure to store the places for quick checking and evenly distributing places within range of 30 miles. Use a minheap to get 10 most cool spots nearby based on their temperature.


Static content accelerator

A library designed for reducing the db query operations and optimizing speed for fetching static tabular data. Inspired by the practical observation that for a table view showing data at front end that is hardly changed, people usually browse first several pages of the table and skip the rest. A sliding window algorithm is used to fetch and cache a portion of data in memory rather than fetching and returning all the tabular data at a time.


Stock market searching (web and ios version)

Stock symbol lookup, current and historical stock data information fetching using public API. Adopted Bootstrap and Jquery UI to achieve a smooth user experience. Including features of symbol autocompleting, Facebook sharing, data auto-refreshing, favoritelist HTML5 localstorage, historical data visualization using Highcharts and related news displaying using Bing News API.


A built-in online shopping system for a financing platform

Developed features including listing of items for sale, retrieving, order generation, cart system, payment for users. Developed features including item creating, updating, deleting, log browsing, order administration at administrators side.


Stock data analysis system

Download stock data of two chinese stock exchange using API provided by yahoo finance to local database. Query the stock with different features including finding most similiar stock curve to the certain one using Pearson product-moment correlation coefficient; Compute k day average of a stock and send to the front end using adobe flex to be displayed.


A web crawler

It takes a list of seeds URLs, parse them to get the link in the webpage source. Push new URLs into the queue. Use a BFS like algorithm to start new round.Case are handled that there may be a infinite loop trap by avoiding duplicate visited pages by maintain a hashtable.


A FAQ management system

An tool for company's internal use. Staffs in customer service section add the asked questions, event processing log and customer information to the system. The section manager create classification as needed. All the FAQs are displayed according to the order of frequency.


xml file display module

It parses the xml file and and convert its content into html code. Then display with a pop up window.


A interactive zoo map application

Create the database model with spatial datatype in SQl. With the information of object position in a zoo, display, query, delete objects according to their spatial correlations. It uses oracle 11g and java with JDBC.
